> **eBookSender** — это Android-приложение для тех, кто читает с электронных книг (в первую очередь **PocketBook**) и хочет закидывать на них файлы напрямую по FTP, минуя облака, провода и сторонние сервисы. Подключился к читалке по Wi-Fi — и отправляй книги, документы и мангу в один тап.

## ✨ Возможности

### 📖 Книги и документы
- **Отправка по FTP без проводов** — через Wi-Fi прямо на устройство.
- **Поддержка популярных форматов**: EPUB, FB2, MOBI, AZW3, PDF, DJVU, DOC/DOCX, CBR/CBZ.
- **Автоопределение устройства**: PocketBook распознаётся автоматически, остальные FTP-серверы — как универсальные.
- **Автоматическая раскладка по папкам**:
  - Книги → `Книги/<Автор>/<Название>.<ext>`
  - Документы → `Документы/<Тег>/<Название>.<ext>`
  - Манга → `Манга/<Серия>/<Том>.<ext>`
- **Извлечение метаданных**: название, автор, серия, год, обложка — подтягиваются автоматически из самого файла (EPUB, FB2, MOBI, PDF, CBZ).
- **Гибкие шаблоны имён** в настройках: `{title}`, `{author}`, `{series}`, `{volume}`, `{year}` и др.
- **Атомарная загрузка**: файлы загружаются через временный `.uploading` и переименовываются только после успешной передачи — никаких «битых» книг на устройстве.

### 🗂️ Каталог устройства
- Просмотр содержимого читалки **без скачивания книг** — прямо из приложения.
- Группировка по авторам, тегам и сериям.
- Для PocketBook — прогресс чтения и статус «прочитано» из базы устройства.
- Удаление файлов прямо из каталога с подтверждением.

### 🌐 Веб-каталоги (OPDS)
- Подключение к онлайн-библиотекам по протоколу **OPDS** (например, Флибуста).
- Поиск по каталогу, обложки, загрузка книг в очередь — **всё в одном месте**.
- Безопасное хранение учётных данных через **Android Keystore**.

### 🧬 Манга (Com-X)
- Встроенный поиск и загрузка глав манги с источника **Com-X**.
- Главы автоматически упаковываются в **CBZ** с прописанным `ComicInfo.xml` (серия, том, название).
- Сохранение серий, история загруженных глав и отметки уже обработанных глав.

### 🎨 Интерфейс и удобство
- Современный дизайн на **Jetpack Compose** и **Material 3 / Material You** (с динамическими цветами на Android 12+).
- Плавные анимации, тёмная и светлая темы.
- Интерфейс на **русском** и **английском** языках, плюс пользовательские переводы через JSON: используйте [пустой шаблон](docs/locales/translation-template.json) или [английский пример](app/src/main/assets/locales/en.json), затем положите готовый `.json` в `eBookSender/locales/` на устройстве.
- Передача работает в **фоновом режиме** — можно закрыть приложение, загрузка продолжится.
- Подключение через **QR-код** или ввод FTP-адреса вручную.

## 📲 Установка для пользователей (простой способ)

Установить приложение на телефон можно за пару минут — **без компьютера и без программирования**.

### Шаг 1. Скачайте установочный файл

1. Перейдите на страницу **[Releases](../../releases)** этого репозитория (вкладка справа или по ссылке).
2. Найдите последний релиз и скачайте файл с расширением **`.apk`** — например:
   ```
   ebookSender-v0.1.0.apk
   ```
   > Если есть файлы с архитектурами (`arm64-v8a`, `armeabi-v7a` и т. д.) — выбирайте **`arm64-v8a`** (подходит почти для всех современных телефонов). Если версия «universal» — берите её.

### Шаг 2. Разрешите установку из неизвестных источников

Android по умолчанию блокирует APK не из Google Play. При первой попытке установки появится подсказка — разрешите установку для вашего браузера или файлового менеджера.

- **Android 8+**: Настройки → Приложения → special access → Установка неизвестных приложений.
- Либо просто подтвердите во всплывающем окне при открытии APK.

### Шаг 3. Установите и запустите

1. Откройте скачанный `.apk` (обычно через «Файлы» или «Менеджер файлов»).
2. Нажмите **Установить** → **Открыть**.
3. Готово! 🎉

### Шаг 4. Подключите читалку

1. Убедитесь, что телефон и читалка **в одной Wi-Fi сети**.
2. На читалке включите FTP-сервер (для PocketBook: *Настройки → Сервисы и протоколы → FTP*).
3. В eBookSender отсканируйте **QR-код** или введите FTP-адрес вручную.
4. Нажмите **«Подключить»** — и можно отправлять файлы!

> 💡 **Подсказка:** после первого подключения устройство сохраняется, и повторное подключение займёт секунду.

## 🛠️ Сборка для разработчиков (продвинутый способ)

Хотите собрать приложение самостоятельно, внести изменения или поучаствовать в разработке? Раздел для вас.

### Требования

| Компонент | Версия |
|---|---|
| **Android Studio** | Hedgehog (2023.1) или новее |
| **Android SDK Platform** | 36 |
| **Android SDK Build Tools** | 34+ |
| **JDK** | 17+ |
| **Gradle** | 8.13 (через `gradlew` wrapper — устанавливать отдельно не нужно) |

### Клонирование репозитория

```sh
git clone https://github.com/<ваш-форк>/PocketBook-SFTP.git
cd PocketBook-SFTP/android
```

### Сборка через Gradle

**Отладочная (debug) сборка:**

```sh
./gradlew :app:assembleDebug
# Результат: app/build/outputs/apk/debug/app-debug.apk
```

**Релизная (release) сборка** — оптимизированная и подписанная:

```sh
./gradlew :app:assembleRelease
# Результат: app/build/outputs/apk/release/app-release.apk
```

> ⚠️ Для релизной сборки нужны параметры подписи. Добавьте их в `local.properties` (этот файл игнорируется git):
> ```properties
> RELEASE_STORE_FILE=release.keystore
> RELEASE_STORE_PASSWORD=<пароль от хранилища>
> RELEASE_KEY_ALIAS=<алиас ключа>
> RELEASE_KEY_PASSWORD=<пароль от ключа>
> ```
> **Никогда не коммитьте пароли и keystore-файлы в репозиторий.**

### Установка на подключённый телефон

```sh
# Отладочная версия
adb install -r app/build/outputs/apk/debug/app-debug.apk

# Релизная версия (предпочтительно)
./gradlew :app:installRelease
```

### Открытие в Android Studio

1. Запустите Android Studio.
2. **File → Open** → выберите папку `android/`.
3. Дождитесь синхронизации Gradle.
4. Выберите конфигурацию **app** и нажмите ▶️ **Run**.

### Песочница / CI

В изолированных окружениях Gradle может потребовать отдельный домашний каталог:

```sh
GRADLE_USER_HOME=/tmp/gradle-home ./gradlew :app:assembleDebug
```
