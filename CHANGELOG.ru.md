# Список изменений

В этом файле описываются заметные для пользователей изменения eBookSender.

## [Unreleased]

### Добавлено

- Добавлен локализованный список изменений прямо в окне новой версии приложения.
- Добавлена автоматическая проверка обновлений сервера PocketBook при подключении PocketBook с общим окном обновления и поддержкой локализованного списка изменений.
- Добавлена очистка кеша списков изменений вместе с кешем APK обновлений.

### Исправлено

- Отключено резервное копирование данных приложения и глобальное разрешение HTTP без шифрования; локальный HTTP-канал управления PocketBook оставлен только для приватных/локальных адресов устройства.
- Скрыт переключатель динамических цветов на устройствах с Android 11 и ниже, так как они не поддерживают динамическую цветовую схему.
- Исправлены подсказки токенов в правилах именования: для книг, документов и манги теперь показываются только подходящие токены.
- Исправлено дублирование индикаторов загрузки при удалении выбранных книг из каталога.
- Скрыт вводящий в заблуждение статус «Не начато» для книг в каталоге обычного FTP-устройства, где прогресс чтения недоступен.
- Исправлены уведомления прогресса отправки и скачивания манги: теперь они остаются видимыми всю фоновую пачку и не пропадают между главами или файлами, но по-прежнему исчезают при возвращении в приложение.
- Исправлено устаревшее состояние FTP-подключения: приложение периодически проверяет устройство и очищает каталог/статус, если сервер больше недоступен.
