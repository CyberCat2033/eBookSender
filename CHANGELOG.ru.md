# Список изменений

В этом файле описываются заметные для пользователей изменения eBookSender.

## [Unreleased]

### Исправлено

- Изменена проверка завершения обновления сервера PocketBook: после доставки launcher-файла приложение показывает окно с просьбой перезапустить pb-ftp, так как текущий запущенный сеанс может продолжать сообщать старую версию до закрытия и повторного открытия.
- Улучшены русские формулировки для обновлений сервера PocketBook, статусов отправки на устройство и сообщений очереди.

## [0.11.7] - 2026-06-21

### Добавлено

- Добавлен отказ от ответственности по гарантиям, ответственности пользователя, сторонним источникам, безопасности FTP, правам на контент и резервному копированию данных.

### Исправлено

- Карточки групп в каталоге теперь раскрываются нажатием по всей поверхности карточки, а не только по кнопке-стрелке.
- Длинные однострочные подписи прогресса, очереди и каталога теперь прокручиваются вместо обрезания.

## [0.11.6] - 2026-06-20

### Исправлено

- Исправлено обновление сервера PocketBook: приложение использует прямую замену launcher-файла, если endpoint активации на устройстве отклонил уже проверенное обновление.
- Исправлены уведомления прогресса загрузки манги: активная фоновая загрузка сохраняет стабильное foreground-уведомление после сворачивания приложения.

## [0.11.5] - 2026-06-20

### Добавлено

- Добавлен локализованный список изменений прямо в окне новой версии приложения.
- Добавлена автоматическая проверка обновлений сервера PocketBook при подключении PocketBook с общим окном обновления и поддержкой локализованного списка изменений.
- Добавлена очистка кеша списков изменений вместе с кешем APK обновлений.
- Добавлен двуязычный список изменений на русском и английском на страницы GitHub Releases для текущей версии приложения.

### Исправлено

- Исправлена публикация списка изменений приложения: клиенты, которые проверили релиз до обновления, теперь видят заметки 0.11.5 вместо пустого раздела «Unreleased».
- Исправлена кнопка проверки обновлений в настройках: тихая проверка при запуске приложения больше не оставляет её неактивной с текстом «Проверка обновлений».
- Убрано название главы из уведомления о скачивании манги, чтобы текст не дублировался.
- Оптимизирована загрузка глав манги: страницы теперь сразу записываются на диск вместо кэширования всех байтов в оперативной памяти, что снижает нагрузку на RAM и предотвращает падения из-за нехватки памяти (OOM).
- Обеспечено удаление пустых и частично загруженных файлов манги `.cbz` при отмене или сбое архивации.
- Устранено состояние гонки при параллельной загрузке и обновлении каталога файлов.
- Реализован экспоненциальный откат (exponential backoff) при повторных попытках скачивания страниц манги для снижения нагрузки на сервер.
- Оптимизирована потокобезопасность менеджера подключений устройства с переходом на неблокирующие обновления StateFlow вместо синхронных блокировок JVM.
- Отключено резервное копирование данных приложения и глобальное разрешение HTTP без шифрования; локальный HTTP-канал управления PocketBook оставлен только для приватных/локальных адресов устройства.
- Скрыт переключатель динамических цветов на устройствах с Android 11 и ниже, так как они не поддерживают динамическую цветовую схему.
- Исправлены подсказки токенов в правилах именования: для книг, документов и манги теперь показываются только подходящие токены.
- Исправлено дублирование индикаторов загрузки при удалении выбранных книг из каталога.
- Скрыт вводящий в заблуждение статус «Не начато» для книг в каталоге обычного FTP-устройства, где прогресс чтения недоступен.
- Исправлены уведомления прогресса отправки и скачивания манги: теперь они остаются видимыми всю фоновую пачку и не пропадают между главами или файлами, но по-прежнему исчезают при возвращении в приложение.
- Исправлено устаревшее состояние FTP-подключения: приложение периодически проверяет устройство и очищает каталог/статус, если сервер больше недоступен.
- Исправлено обновление сервера PocketBook: приложение больше не сообщает об успехе, пока обновлённый сервер реально не вернул новую версию.
- Исправлен staging при обновлении сервера PocketBook: загружаемый launcher теперь сохраняет финальное имя `pb-ftp.app` вместо имени с префиксом версии.
